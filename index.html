📘 HTML - Linguagem de Marcação de Hipertexto

O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Ele descreve a estrutura do conteúdo usando "marcações" (tags). Pense no HTML como o esqueleto de um site.

🔹 Tags

Elementos que definem o conteúdo. Ex: <h1>Título</h1>

🔹 Atributos

Propriedades que modificam as tags. Ex: <img src="foto.jpg">

🔹 Estrutura

Documento HTML tem <html>, <head> e <body>

🎯 Exemplo Prático de HTML

Código HTML:

<div class="card">
    <h3>💡 Card de Exemplo</h3>
    <p>Este é um parágrafo dentro de um card</p>
    <ul>
        <li>Item 1 da lista</li>
        <li>Item 2 da lista</li>
    </ul>
</div>

Resultado Renderizado:

💡 Card de Exemplo

Este é um parágrafo dentro de um card

  • Item 1 da lista
  • Item 2 da lista

✅ O HTML define a estrutura - o que você vê acima é o resultado direto do código HTML ao lado.

🎨 CSS - Folhas de Estilo em Cascata

O que é CSS?

CSS (Cascading Style Sheets) é a linguagem usada para estilizar o HTML. Enquanto o HTML é o esqueleto, o CSS é a aparência - define cores, tamanhos, fontes e posicionamento.

🔹 Seletores

Identificam quais elementos estilizar. Ex: p, .classe, #id

🔹 Propriedades

Características que você altera. Ex: color, background

🔹 Cascata

A ordem das regras importa - estilos podem ser herdados

⚡ JavaScript - A Linguagem da Interatividade

O que é JavaScript?

JavaScript é a linguagem de programação que adiciona comportamento e interatividade às páginas web. Se HTML é o esqueleto e CSS é a aparência, JavaScript são os músculos - faz o site responder a cliques e interagir com o usuário.

🔹 Eventos

Ações do usuário: click, mouseover

🔹 DOM

Manipulação dos elementos HTML via JavaScript

🔹 Funções

Blocos de código que executam tarefas específicas

Como HTML, CSS e JavaScript trabalham juntos?

🎨 CSS

Estiliza a aparência (cores, tamanhos, animações)

⚡ JavaScript

Adiciona interatividade (responder a cliques, processar dados)
