📘 HTML - Linguagem de Marcação de Hipertexto

O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Ele descreve a estrutura do conteúdo usando "marcações" (tags). Pense no HTML como o esqueleto de um site - ele define onde cada elemento (títulos, parágrafos, imagens, links) deve ficar.

🔹 Tags

São elementos que definem o conteúdo. Ex: <h1>Título</h1>, <p>Parágrafo</p>

🔹 Atributos

Propriedades que modificam as tags. Ex: <img src="foto.jpg" alt="Descrição">

🔹 Estrutura

Documento HTML tem <html>, <head> e <body>

🎯 Exemplo Prático de HTML

Código HTML:

<div class="card">
    <h3>💡 Card de Exemplo</h3>
    <p>Este é um parágrafo dentro de um card</p>
    <ul>
        <li>Item 1 da lista</li>
        <li>Item 2 da lista</li>
    </ul>
</div>

🎨 CSS - Folhas de Estilo em Cascata

O que é CSS?

CSS (Cascading Style Sheets) é a linguagem usada para estilizar o HTML. Enquanto o HTML é o esqueleto, o CSS é a pele, roupas e maquiagem - define cores, tamanhos, fontes, posicionamento e animações.

🔹 Seletores

Identificam quais elementos estilizar. Ex: p (tag), .classe, #id

🔹 Propriedades

Características que você altera. Ex: color, background, font-size

🔹 Cascata

A ordem das regras importa - estilos podem ser herdados e sobrepostos

🎯 Exemplo Prático de CSS

Elemento sem estilo

Clique no botão para ver o CSS em ação

CSS aplicado ao clicar:

{
    background: linear-gradient(135deg, #667eea, #764ba2);
    color: white;
    border: none;
    box-shadow: 0 10px 20px rgba(102, 126, 234, 0.3);
    transform: scale(1.1);
    transition: all 0.3s ease;
}

✅ O CSS transforma a aparência - observe como o elemento muda quando você aplica os estilos.

⚡ JavaScript - A Linguagem da Interatividade

O que é JavaScript?

JavaScript é a linguagem de programação que adiciona comportamento e interatividade às páginas web. Se HTML é o esqueleto e CSS é a aparência, JavaScript são os músculos e o cérebro - faz o site responder a cliques, validar formulários, criar animações, etc.

🔹 Eventos

Ações do usuário: click, mouseover, keypress

🔹 DOM

Manipulação dos elementos HTML via JavaScript

🔹 Funções

Blocos de código que executam tarefas específicas

🧪 Área Prática - Combine Tudo!

Como HTML, CSS e JavaScript trabalham juntos?

Os três formam a base do desenvolvimento web moderno:

📘 HTML

Cria a estrutura dos elementos (inputs, botões, textos)

🎨 CSS

Estiliza a aparência (cores, tamanhos, animações)

⚡ JavaScript

Adiciona interatividade (responder a cliques, processar dados)
